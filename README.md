# QA Checker 크롬 익스텐션

개발자도구(F12)에 QA Checker 탭을 추가해서 속성 값 누락, 루비콘 JSON 오류, 이벤트 번호 누락, 앵커드URL 오류를 검사하는 익스텐션.

자세한 구조는 [폴더_구조_설명.txt](./폴더_구조_설명.txt) 참고.

## 설치 (개발자 모드로 로드하기)

1. 이 저장소를 클론하거나 다운로드한다.
2. Chrome 주소창에 `chrome://extensions` 입력.
3. 우측 상단 "개발자 모드" 켜기.
4. "압축해제된 확장 프로그램을 로드합니다" 클릭 → 이 폴더(`manifest.json`이 있는 위치) 선택.
5. 아무 페이지에서나 F12 → "QA Checker" 탭 확인.

코드 수정 후에는 `chrome://extensions`에서 새로고침 버튼만 누르면 반영된다.
